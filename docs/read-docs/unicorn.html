I AM WHAT I CODE MAKE READ
서버 입문 기록
  2021.08.03. 화요일

  그러니까, 나는 여태까지 '서버'라는 것과 인연이 없었다. 자료구조같은 기본적인 공부와 함께 졸업에 필요한 과목들부터 순차적으로 배워나가다보니 서버에는 관심을 가질 틈도 없었다. 공부를 해야 하는 계기도, 당장의 필요도 없었던 내 관심을 서버로 끌었던 건 다른 게 아니라 분홍색 유니콘 한 마리 였다.

  작년부터 웹코딩에 관심을 가지고 방학때마다 조금씩 html과 css, javascript를 공부하고 있었다. 간단한 웹페이지를 하나 구상하고서 내 구상을 실현시킬 수 있는 코드를 구글에 찾아보는 식으로 주먹구구 공부했다. 조악하게나마 만든 웹페이지는 Github pages를 이용해서 호스팅을 하고 배포했다. 덕분에 내 Github repository는 각종 웹페이지로 채워지기 시작했다. 제대로 된 짜임새를 만들 줄 몰라 일단 내용을 떄려박기에 급급했던 첫 웹페이지에 비하면 지금 만들고 있는 이 페이지는 제법 그럴싸 해졌다고 생각한다.

  지금 만들고 있는 바로 이 웹페이지를 Github에 호스팅하고 잘 접속되는지 확인해보려는 찰나였다. 주소창에 내 웹페이지의 주소를 입력하고 Enter 키를 의기양양하게 누른 순간, 나는 처음 보는 에러와 맞닥뜨렸다. 503 Error라는 메세지와 함께 나는 사나운 유니콘과 눈이 마주쳤다.

 

  인터넷에 찾아보니 503 에러는 해당 페이지가 unavailable할 때 발생한다고 한다.
